# refs/

Scratch workspace for running `pdf2md.py` against real PDFs. Drop
a PDF here and run:

```
./pdf2md.py refs/<name>.pdf refs/<name>.md
```

Everything in this directory is gitignored except this README —
transcribed `.md` files and source PDFs stay local.

See [`../refs-transcription-protocol.md`](../refs-transcription-protocol.md)
for the full per-PDF procedure including the vision-fallback path.
